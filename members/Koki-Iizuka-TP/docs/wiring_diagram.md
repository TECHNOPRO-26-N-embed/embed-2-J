## 配線図仕様まとめ（コード形式）

---

### 主要素

```
- Arduino Uno（または互換マイコン）
- ボタン1（再生/停止）：PIN 2（INPUT_PULLUP）
- ボタン2（曲選択）　　：PIN 3（INPUT_PULLUP）
- LED（状態表示）　　　：PIN 9（出力、330Ω直列推奨）
- パッシブブザー　　　 ：PIN 10（出力、330Ω直列推奨）
- I2C液晶（16x2等）　　：SDA=A4, SCL=A5
- 電源（5V/GND）
```

---

### 回路テキスト図（簡易）

```
[Arduino Uno/Nano]
 ├─ 2  ──[タクトスイッチ]──┬── GND      // ボタン1
 ├─ 3  ──[タクトスイッチ]──┴── GND      // ボタン2
 ├─ 9  ──[330Ω]────>|────── GND         // LED（+＝アノード）
 ├─10  ──[330Ω]───[パッシブブザー]── GND // ブザー
 ├─A4  ──SDA────┐
 ├─A5  ──SCL───┬┘                         // I2C液晶
 ├─5V  ──[I2C液晶モジュールVCC]
 └─GND ──[I2C液晶GND]
```

---

### ピン配置表

| Arduinoピン | 用途         | 接続先           | 備考               |
|:------------|:-------------|:-----------------|:-------------------|
| 2           | ボタン1      | タクトS1, GND    | INPUT_PULLUP有効   |
| 3           | ボタン2      | タクトS2, GND    | INPUT_PULLUP有効   |
| 9           | LED          | LED+（アノード）, 330Ω→GND | 出力、点灯用       |
| 10          | ブザー       | パッシブブザー, 330Ω→GND  | 音出力             |
| A4          | SDA          | I2C液晶SDA端子   | 16x2等I2C液晶      |
| A5          | SCL          | I2C液晶SCL端子   |                    |
| 5V          | 電源         | I2C液晶VCC       | センサ/LCD等も共用 |
| GND         | グランド     | 全デバイスGND    | 共通グランド       |

---

### 部品一覧

```
- Arduino UnoまたはNano：1台
- タクトスイッチ：2個（曲選択・再生停止用）
- LED（3mm〜5mm）：1個（出力状態表示用）
- 330Ω抵抗：2本（LED・ブザーそれぞれに直列）
- パッシブブザー：1個（音楽再生用）
- I2C液晶モジュール（16×2等）：1個（表示用、I2Cタイプ）
- ジャンパワイヤ、ブレッドボード等
```

---

### 注意点

```
- ボタンは「INPUT_PULLUP」で使うので、外部抵抗は原則不要。
- LED・ブザーには必ず330Ω程度の抵抗を直列に入れる（過電流防止）。
- I2C液晶はA4（SDA）、A5（SCL）で接続。VCC/GNDもお忘れなく。
- 全デバイスのGNDをArduinoのGNDにつなぐことで共通グラウンドとする。
- 各部品の向き（LEDのアノード/カソードなど）に注意。
```
